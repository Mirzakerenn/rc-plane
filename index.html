Fase 1 — Persiapan Software
Di PC/laptop kamu dulu, sebelum sentuh hardware
1
Download & Install Arduino IDE
~10 menit · Software gratis
▾

Arduino IDE adalah aplikasi untuk menulis dan mengupload kode ke Arduino Nano kamu.

  1. Buka browser, pergi ke arduino.cc/en/software
  2. Klik "Download Arduino IDE 2" → pilih Windows/Mac sesuai PC kamu
  3. Jalankan installer, klik Next terus sampai selesai
  4. Buka Arduino IDE setelah install selesai
✅ Kalau sudah terbuka dan ada layar editor putih kosong, berarti berhasil.
2
Install Driver CH340 (untuk Arduino Nano "murah")
~5 menit · Wajib kalau Arduino Nano kamu CH340
▾

Arduino Nano v3 yang pakai chip CH340 butuh driver tambahan agar PC bisa mengenalinya.

  1. Google: "CH340 driver download"
  2. Download dari situs wch.cn atau sparks.gogo.mn/ch340.html
  3. Extract ZIP, jalankan SETUP.EXE, klik Install
  4. Colokkan Arduino Nano ke PC via kabel USB
  5. Buka Device Manager → cek apakah ada "USB-SERIAL CH340 (COMx)"
⚠️ Kalau di Device Manager masih ada tanda seru kuning, coba restart PC dulu setelah install driver.
3
Install Library MPU6050_tockn di Arduino IDE
~3 menit · Library untuk sensor IMU
▾

Library adalah kumpulan kode siap pakai. Kita butuh library khusus untuk membaca sensor MPU6050.

  1. Buka Arduino IDE
  2. Klik menu Tools → Manage Libraries... (atau tekan Ctrl+Shift+I)
  3. Di kotak search, ketik: MPU6050_tockn
  4. Klik library bernama "MPU6050_tockn by tockn"
  5. Klik tombol Install
  6. Tunggu sampai muncul tulisan "INSTALLED"
ℹ️ Library Wire dan Servo sudah otomatis ada di Arduino IDE, tidak perlu install manual.
4
Upload Kode ke Arduino Nano
~5 menit · Inti dari fase ini
▾

Sekarang waktunya memasukkan kode yang sudah dibuat ke Arduino.

  1. Buka file rc_plane_controller.ino yang sudah didownload (klik dua kali)
  2. Di Arduino IDE, pilih menu Tools → Board → Arduino AVR Boards → Arduino Nano
  3. Pilih menu Tools → Processor → ATmega328P (Old Bootloader) (pilih ini kalau upload gagal)
  4. Pilih menu Tools → Port → COMx (pilih COM yang ada "CH340"-nya)
  5. Klik tombol → Upload (ikon panah ke kanan, atau tekan Ctrl+U)
  6. Tunggu — di bawah layar akan muncul "Done uploading."
🚫 Kalau upload error "avrdude: stk500...", coba ganti Processor ke "ATmega328P (Old Bootloader)" di langkah 3.
✅ Upload berhasil = LED kecil di Arduino Nano akan berkedip sebentar lalu berhenti. Normal!

Fase 2 — Merangkai Hardware
Sambungkan semua komponen ke Arduino Nano
5
Sambungkan MPU6050 (Sensor Gyro/Accelerometer)
Protokol I2C — 4 kabel
▾

MPU6050 adalah sensor yang mendeteksi kemiringan pesawat (roll & pitch).

Pin MPU6050→Pin Arduino NanoKabel
VCC→3.3VMerah
GND→GNDHitam
SDA→A4Biru
SCL→A5Kuning
⚠️ VCC MPU6050 ke pin 3.3V, BUKAN 5V. Kalau salah, sensor bisa rusak permanen.
6
Sambungkan 2 Servo SG90 (Aileron & Elevator)
Sinyal PWM — masing-masing 3 kabel
▾

Servo SG90 punya 3 kabel: Cokelat (GND), Merah (VCC 5V), Oranye/Kuning (Sinyal).

ServoKabel Servo→Pin Arduino
AileronOranye (sinyal)→D3
AileronMerah (VCC)→5V
AileronCokelat (GND)→GND
ElevatorOranye (sinyal)→D5
ElevatorMerah (VCC)→5V
ElevatorCokelat (GND)→GND
ℹ️ Pin 5V dan GND Arduino bisa dibagi (parallel) untuk dua servo sekaligus pakai breadboard.
7
Sambungkan ESC + Motor Brushless
⚡ Bagian dengan tegangan tinggi — hati-hati
▾

ESC (Electronic Speed Controller) mengontrol motor brushless. ESC punya 2 sisi koneksi:

Sisi INPUT (dari baterai LiPo):

  • Kabel merah tebal ESC → kutub + baterai
  • Kabel hitam tebal ESC → kutub − baterai

Sisi OUTPUT ke Arduino (kabel tipis 3-pin):

Kabel ESC→Pin Arduino
Kuning/Putih (sinyal)→D9
Merah (opsional BEC)→Biarkan / tidak perlu
Hitam (GND)→GND
🚫 JANGAN sambungkan baterai LiPo dulu sampai semua kabel sudah benar dan kode sudah diupload. Baling-baling JANGAN dipasang saat testing di meja.
8
Sambungkan Bluetooth HC-05/HC-06
Serial UART — 4 kabel
▾

HC-05/HC-06 adalah modul Bluetooth yang menerima perintah dari HP kamu.

Pin HC-05/06→Pin Arduino Nano
VCC→5V
GND→GND
TX→RX (D0)
RX→TX (D1)
⚠️ TX HC-05 → RX Arduino, dan RX HC-05 → TX Arduino (silang/cross). Kalau terbalik, komunikasi tidak akan jalan.
💡 Lepas kabel TX/RX HC-05 dulu saat upload kode. Pasang kembali setelah upload selesai.

Fase 3 — Setup Aplikasi HP
Hubungkan HP ke Arduino via Bluetooth
9
Install "Arduino Bluetooth Controller" di HP Android
~2 menit · Gratis di Play Store
▾
  1. Buka Play Store di HP Android
  2. Search: Arduino Bluetooth Controller
  3. Install aplikasi yang ikonnya mirip logo Arduino (warna biru/hijau)
  4. Nyalakan Bluetooth di HP kamu
  5. Pair dengan HC-05 atau HC-06 (PIN default: 1234 atau 0000)
  6. Buka aplikasinya, pilih perangkat HC-05/06, pilih mode Terminal
ℹ️ Mode Terminal memungkinkan kamu kirim karakter satu per satu. Ini yang dipakai kode kita.
10
Test Pertama — Tanpa Motor, Tanpa Baling-baling!
Verifikasi semua komponen berfungsi
▾

Sebelum terbang, test di meja dulu. Ini daftar perintah yang bisa kamu kirim dari HP:

W
Throttle naik
S
Throttle turun
A
Roll kiri
D
Roll kanan
U
Pitch naik
J
Pitch turun
X
ARM / DISARM
R
Reset PID

Checklist test:

  • ☐ Kirim 'A' → servo aileron bergerak ke kiri
  • ☐ Kirim 'D' → servo aileron bergerak ke kanan
  • ☐ Kirim 'U' → servo elevator bergerak ke atas
  • ☐ Kirim 'J' → servo elevator bergerak ke bawah
  • ☐ Miringkan papan Arduino → servo bergerak otomatis untuk koreksi
🚫 Test motor (kirim 'X' lalu 'W') hanya kalau motor sudah terpasang aman dan TANPA baling-baling. Baling-baling bisa menyebabkan cedera serius.

Fase 4 — Tuning PID
Setelah hardware berjalan normal, lakukan ini sebelum terbang
11
Aktifkan Debug Mode & Lihat Data Sensor
Pahami dulu data yang keluar
▾

Buka kode, cari bagian debug di fungsi computeAndWrite(), hapus tanda /* dan */:

// Sebelum (debug mati): /* Serial.print("R:"); Serial.print(currentRoll, 1); ... */ // Sesudah (debug aktif) — hapus /* dan */: Serial.print("R:"); Serial.print(currentRoll, 1); Serial.print(" P:"); Serial.print(currentPitch, 1);

Upload ulang kode, lalu buka Tools → Serial Monitor (Ctrl+Shift+M), set baud ke 9600. Kamu akan lihat data real-time.

12
Tuning Nilai Kp, Ki, Kd
Lakukan bertahap, satu nilai per satu
▾

Cari baris ini di kode dan ubah nilainya:

PIDGains gainRoll = { 2.5f, 0.02f, 0.8f }; // ↑ Kp ↑ Ki ↑ Kd PIDGains gainPitch = { 2.5f, 0.02f, 0.8f };

Urutan tuning:

  1. Set semua ke 0: { 0, 0, 0 }
  2. Naikkan Kp dari 1.0 sampai servo mulai gemetar/osilasi, lalu turunkan sedikit
  3. Naikkan Kd dari 0.1 sampai osilasi hilang (biasanya sekitar 1/3 dari Kp)
  4. Naikkan Ki dari 0.01 perlahan — stop kalau ada getaran lambat

Diagnosis masalah:

GejalaSolusi
Servo jitter cepatTurunkan Kd
Pesawat goyang lambatTurunkan Ki
Koreksi lambatNaikkan Kp
Servo full defleksiCek orientasi MPU6050, tambah * -1 ke getAngleX()
